# Python旅游景点推荐系统 python551

## 项目概述

本项目是一款基于Python+Flask+vue+MySQL的旅游景点推荐系统。系统包含首页、个人中心、用户管理、景点信息管理、景点类型管理、景点门票管理、在线反馈和系统管理等多个功能模块，旨在为用户提供便捷的旅游信息管理与推荐服务。

## 技术栈

- 编程语言：Python
- 开发工具：Pycharm
- 后端框架：Flask
- 前端框架：vue
- 数据库版本：MySQL 5.7+
- 数据库工具：Navicat、SQLyog

## 功能模块

- 系统首页
- 个人中心
- 用户管理
- 景点信息管理
- 景点类型管理
- 景点门票管理
- 在线反馈
- 系统管理

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Python环境
- Flask框架
- vue框架
- MySQL数据库

## 部署步骤

1. 环境准备：确保Python、Flask、vue、MySQL等环境正确安装和配置。
2. 数据库搭建：使用Navicat或SQLyog创建数据库，并导入相关表结构及数据。
3. 后端部署：将项目后端代码部署至服务器，并配置相关环境变量。
4. 前端部署：将项目前端代码部署至Web服务器。
5. 系统测试：进行功能测试，确保系统正常运行。
